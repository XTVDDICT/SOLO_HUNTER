# SOLO_HUNTER

A DigiByte wallet display for the ESP32-2432S028 "Cheap Yellow Display" (CYD).

It shows a neon-style Bitcoin image, live DigiByte wallet balance fetched directly on the device, a green border, and a simple BLOCK FOUND popup when the wallet balance increases.

## Setup

Edit these lines before uploading:

```cpp
const char* WIFI_SSID = "YOUR_WIFI_SSID";
const char* WIFI_PASS = "YOUR_WIFI_PASSWORD";
const char* WALLET = "YOUR_DGB_WALLET_ADDRESS";
