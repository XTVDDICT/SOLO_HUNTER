# SOLO_HUNTER
ESP32 CYD (Cheap Yellow Display) DigiByte wallet monitor that runs fully on-device. Displays a custom neon-style Bitcoin image, live wallet balance via Chainz API, and a “BLOCK FOUND” popup when balance increases. Clean landscape layout with a green border, no PC relay required.
